## 1. Level Overview

### 1.1 Curriculum Alignment

| Aspect | Detail |
|--------|--------|
| French level | CE1 -- Cycle 2, Year 2 |
| French reference | "Attendus de fin de CE1" from BO 2024; fraction additions from BO 2025 |
| UK alignment (lower SLs) | Year 2 (KS1) -- numbers to 100, addition/subtraction facts to 20, x2/x5/x10 tables |
| UK alignment (upper SLs) | Year 3 (lower KS2) -- numbers to 1000, columnar addition/subtraction, x3/x4/x8 tables |
| Age | 7 (turning 8 during the school year) |

### 1.2 Key Objectives

- **Numbers:** Read, write, compare, order, decompose numbers to 999
- **Addition:** Written addition with carrying, up to 3-digit numbers
- **Subtraction:** Written subtraction with borrowing, up to 3-digit numbers
- **Multiplication:** Understand concept of repeated addition; learn tables x2, x3, x4, x5
- **Fractions:** Unit fractions (1/2 through 1/10), non-unit fractions, compare same-denominator fractions, add/subtract same-denominator fractions (2025 programme)
- **Geometry:** Name and describe polygons, right angles introduction, solids identification
- **Measurement:** cm/m, g/kg, hours/minutes, euros/centimes
- **Problem solving:** 1-step and 2-step word problems using all four operations in context

### 1.3 Mental Math Targets (End of CE1)

| Skill | Target |
|-------|--------|
| Addition facts to 20 | Fluent recall |
| Complements to 10 | Automatic |
| Doubles and near-doubles | Automatic |
| Add/subtract 9, 19, 29 | Learned procedure |
| Add/subtract multiples of 10 | Learned |
| Halves of even numbers to 20 | Learned |
| Multiplication tables x2, x3, x4, x5 (both ways) | Known |
| Fluency test | 12 correct results in 3 minutes |

### 1.4 Period-to-Sub-Level Mapping

| Sub-Levels | French Period | Months | Focus |
|------------|--------------|--------|-------|
| SL1-2 | Period 1 (Sept-Oct) | Sept-Oct | Review CP, numbers to 100, addition/subtraction review |
| SL3-4 | Period 2 (Nov-Dec) | Nov-Dec | Numbers to 200, addition with carrying, fractions intro (2025) |
| SL5-6 | Period 3 (Jan-Feb) | Jan-Feb | Numbers to 500, subtraction with borrowing, multiplication x2/x5 |
| SL7-8 | Period 4 (Mar-Apr) | Mar-Apr | Numbers to 999, multiplication x3/x4, measurement units |
| SL9-10 | Period 5 (May-Jun) | May-Jun | Full CE1 + UK Year 3 objectives, all operations, problem solving |

## 2. Topic Definitions

### CE1-NUM: Numbers and Place Value (0-999)

| Aspect | Detail |
|--------|--------|
| Scope | Read, write, compare, order, decompose whole numbers 0-999 |
| Place value | Units, tens, hundreds |
| Representations | Digits, words, base-10 blocks, number line, decomposition (e.g., 345 = 300 + 40 + 5) |
| Counting patterns | By 1s, 2s, 5s, 10s, 100s; forward and backward |
| Comparison | Using <, >, = symbols |

### CE1-ADD: Addition (with carrying, up to 3 digits)

| Aspect | Detail |
|--------|--------|
| Mental addition | Two-digit + one-digit, two-digit + two-digit (no carry) |
| Written addition (no carry) | Column format, 2-digit + 2-digit, then 3-digit + 3-digit |
| Written addition (with carry) | Carrying from units to tens, from tens to hundreds |
| Properties | Commutativity (a + b = b + a), adding zero |
| Range | Operands 0-999, results 0-999 |

### CE1-SUB: Subtraction (with borrowing, up to 3 digits)

| Aspect | Detail |
|--------|--------|
| Mental subtraction | Two-digit - one-digit, two-digit - two-digit (no borrow) |
| Written subtraction (no borrow) | Column format, 2-digit - 2-digit, then 3-digit - 3-digit |
| Written subtraction (with borrow) | Borrowing from tens to units, from hundreds to tens |
| Constraint | Result always >= 0 (a >= b enforced) |
| Range | Operands 0-999, results 0-999 |

### CE1-MUL: Multiplication Introduction

| Aspect | Detail |
|--------|--------|
| Concept | Repeated addition, groups of objects, arrays |
| Tables | x2 (Period 2-3), x5 (Period 3-4), x3 (Period 4), x4 (Period 4-5) |
| UK addition | x10 table (UK Year 2 requirement, not explicitly French CE1 but natural extension) |
| Multiply by 10 | Introduced in Period 5 |
| Notation | a x b = c; understanding commutativity |
| Range | Factors 1-10 (for learned tables), products up to 50 |

### CE1-FRA: Fractions Introduction (2025 Programme)

| Aspect | Detail |
|--------|--------|
| Unit fractions | 1/2, 1/3, 1/4, 1/5, 1/6, 1/8, 1/10 |
| Non-unit fractions | Numerator < denominator (all fractions <= 1) |
| Denominators | 2, 3, 4, 5, 6, 8, 10 |
| Vocabulary | Numerator (numerateur), denominator (denominateur) |
| Comparison | Same-denominator fractions; comparing unit fractions (1/2 > 1/3 > 1/4...) |
| Operations | Addition and subtraction with same denominator (Period 5) |
| Visual | Fraction bars, pie charts, shaded shapes |

### CE1-GEO: Geometry

| Aspect | Detail |
|--------|--------|
| 2D shapes | Circle, square, rectangle, triangle -- naming, describing, reproducing |
| Properties | Number of sides, number of vertices |
| Right angles | Coding for right angles introduced (Period 3), using set square |
| Alignment | Verifying alignment of points |
| 3D solids | Cube, rectangular prism (pave droit), cone, pyramid, cylinder, sphere |
| Solid properties | Faces, vertices, edges (Period 4-5) |
| Tools | Ruler, compass (introduction), set square (equerre) |
| Note | Symmetry is NOT in CE1 in 2025 programme (moved to CE2) |

### CE1-MSR: Measurement

| Aspect | Detail |
|--------|--------|
| Length | cm and m; 1 m = 100 cm; measuring with ruler; measuring broken lines |
| Mass | g and kg; 1 kg = 1000 g; comparing masses |
| Time | AM/PM distinction; hour and minute units; reading analog clock (whole, half, quarter hours) |
| Money | Euro and centime; making amounts; simulating purchases |
| Duration | Comparing elapsed durations |

### CE1-PRB: Problem Solving

| Aspect | Detail |
|--------|--------|
| Structure | 1-step problems (SL1-6), 2-step problems (SL7-10) |
| Operations | Addition, subtraction in all SLs; multiplication in context from SL5; division (sharing) from SL7 |
| Representations | Drawings, schemas, bar models |
| Answer format | Always a single unambiguous numerical answer |

**Scope:** Read, write, compare, order numbers 0-59. Understand tens and units decomposition.

**Example Questions:**

*Micro 1:*
- "Quel nombre vient apres 5 ?" -> 6 [MC: 4, 6, 7]
- "Compare: 3 ___ 3" -> = [MC: <, >, =]
- "Combien y a-t-il de pommes ?" (image: 4 apples) -> 4

*Micro 5:*
- "Decompose 25: ___ dizaines et ___ unites" -> 2 dizaines et 5 unites
- "Range du plus petit au plus grand: 18, 7, 31, 12" -> 7, 12, 18, 31

- "Compare: 47 ___ 54" -> < [MC: <, >, =]

**Common Mistakes for Distractors:**
- Confusing tens and units (writing 52 instead of 25)
- Reversing comparison direction (< instead of >)
- Counting errors (+/- 1 when counting by 2s or 5s)
- Writing numbers phonetically wrong (e.g., "soixante" confusion begins)

**Mastery Criteria:**
- 80% accuracy on rolling 10 questions per micro-level
- Must correctly decompose any number 0-59 into tens and units
- Must correctly compare any pair of numbers 0-59

#### CE1-NUM at SL2: Numbers 0-99

**Scope:** Full two-digit number mastery. Tens and units decomposition. Counting by 2s from 0.

| Parameter | Micro 1 | Micro 3 | Micro 5 | Micro 7 | Micro 10 |
|-----------|---------|---------|---------|---------|----------|
| Number range | 0-69 | 0-79 | 0-89 | 0-99 | 0-99 |

| Comparison complexity | 2 numbers | 2 numbers | 3 numbers (ordering) | 4 numbers (ordering) | 5-6 numbers (ordering) |

| Time per Q (s) | 18 | 16 | 14 | 12 | 10 |

**Note on French number words:** The 70-99 range is particularly challenging in French:
- 70 = soixante-dix (sixty-ten)
- 80 = quatre-vingts (four-twenties)
- 90 = quatre-vingt-dix (four-twenty-ten)

These require explicit practice and appear from micro 3 onward.

**Example Questions:**

*Micro 1:*
- "Ecris en chiffres: soixante-trois" -> 63
- "Decompose 65: ___ dizaines et ___ unites" -> 6, 5
- "Quel nombre vient juste avant 60 ?" -> 59 [MC: 58, 59, 61]

*Micro 5:*
- "Ecris en chiffres: quatre-vingt-deux" -> 82
- "Range du plus petit au plus grand: 73, 89, 41" -> 41, 73, 89
- "Compte de 10 en 10 a partir de 34: 34, ___, ___, ___" -> 44, 54, 64

*Micro 10:*
- "Ecris en lettres: 97" -> quatre-vingt-dix-sept [MC with 4 options]
- "Range du plus petit au plus grand: 91, 19, 78, 87, 9" -> 9, 19, 78, 87, 91
- "Compte de 2 en 2 a rebours depuis 80: 80, ___, ___, ___, ___" -> 78, 76, 74, 72

**Common Mistakes for Distractors:**
- soixante-dix = 60+10 confusion (writing 6010 or 610)
- quatre-vingts = 420 (literal translation error)
- quatre-vingt-dix-sept = 4207 or similar
- Reversing digits when writing from words (79 -> 97)
- Ordering errors with 70-99 range numbers

**Mastery Criteria:**
- 80% accuracy on rolling 10
- Correctly read/write all numbers 0-99 in digits and words (including 70-99 French names)
- Order any set of 4+ numbers within 0-99

- **Adding denominators:** 1/4 + 2/4 = 3/8 (adds both numerators AND denominators)
- **Subtracting denominators:** 3/4 - 1/4 = 2/0

- **Wrong operation:** adding when should subtract
- **Result > 1:** computing 3/4 + 2/4 = 5/4 (but this SL constrains to <= 1)

**Mastery Criteria:**
- 80% accuracy on rolling 10
- Add fractions with same denominator (result <= 1)
- Subtract fractions with same denominator
- Understand that the denominator stays the same when adding/subtracting
- Apply fraction operations in simple word problems
